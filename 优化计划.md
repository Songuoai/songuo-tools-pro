## 🎯 第一步优化：立即执行

### 优化 1：图片懒加载

**目标：** 提升首页加载速度

**实现：**
```tsx
// 使用 Next.js Image 组件
import Image from 'next/image'

<Image
  src={tool.logoUrl}
  alt={tool.name}
  loading="lazy"
  width={48}
  height={48}
  placeholder="blur"
  blurDataURL="data:image/jpeg;base64,..."
/>
```

### 优化 2：骨架屏

**目标：** 提升加载体验

**实现：**
```tsx
// 加载时显示骨架屏
{loading ? (
  <Skeleton className="h-48 w-full" />
) : (
  <ToolCard tool={tool} />
)}
```

### 优化 3：SEO Meta 标签

**目标：** 提升搜索引擎排名

**实现：**
```tsx
// 每页独立的 SEO 配置
<Head>
  <title>{tool.name} - 松果工具箱</title>
  <meta name="description" content={tool.shortDesc} />
  <meta property="og:title" content={tool.name} />
  <meta property="og:image" content={tool.logoUrl} />
</Head>
```
