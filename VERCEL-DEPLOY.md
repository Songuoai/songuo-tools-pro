# 🚀 Vercel 部署配置

**部署时间：** 2026-04-02 13:45
**原因：** Cloudflare Pages DNS 传播太慢，Vercel 更快

---

## Vercel 部署步骤

### 1️⃣ 访问 Vercel
https://vercel.com/new

### 2️⃣ 导入项目
- 点击 "Import Git Repository"
- 选择 Gitee 仓库：`pinecone-tool-station_0/songuo-tools-pro`

### 3️⃣ 配置构建
- **Framework Preset:** Next.js
- **Build Command:** `npm run build`
- **Output Directory:** `out`
- **Install Command:** `npm install`

### 4️⃣ 部署
- 点击 "Deploy"
- 等待 2-3 分钟
- 获得访问地址：`https://songuo-tools-pro.vercel.app`
