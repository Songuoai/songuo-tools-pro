## 🎯 特色功能

### 1. 实时访问人数 ⭐⭐⭐⭐⭐
```javascript
// 每 5 秒自动更新，模拟真实访问
useEffect(() => {
  const interval = setInterval(() => {
    setVisitorCount(prev => prev + Math.floor(Math.random() * 3))
  }, 5000)
  return () => clearInterval(interval)
}, [])
```
