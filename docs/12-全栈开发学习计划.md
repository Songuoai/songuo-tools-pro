## 🛠️ 遇到问题解决方案

### 学习方法论

```
1. 明确问题
   - 错误信息是什么？
   - 预期行为 vs 实际行为
   - 复现步骤

2. 搜索解决方案
   - Google 搜索（英文 + 中文）
   - Stack Overflow
   - GitHub Issues
   - 官方文档

3. 实践验证
   - 尝试解决方案
   - 记录解决过程
   - 总结教训

4. 知识沉淀
   - 更新项目文档
   - 创建学习笔记
   - 分享给他人
```

## 💡 学习技巧

### 1. 刻意练习
```
- 每天至少写 100 行代码
- 每周完成 1 个小项目
- 每月完成 1 个大功能
```

### 2. 费曼学习法
```
- 学习一个概念
- 尝试教会别人（写博客、录视频）
- 发现理解盲区
- 回头重新学习
```

### 3. 项目驱动
```
- 以松果工具箱为实战项目
- 每个功能都是学习机会
