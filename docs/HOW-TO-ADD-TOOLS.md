## 🎯 方式 1：后台添加（推荐）

### 适用场景
- ✅ 日常添加新工具
- ✅ 修改现有工具
- ✅ 删除过时工具

### 操作步骤

#### 步骤 1：登录后台
```
访问：http://localhost:3001/admin/login
账号：admin
密码：123456
```

#### 步骤 2：进入工具管理
```
点击左侧菜单：工具管理
```

#### 步骤 3：添加工具
```
点击页面右上角："添加工具" 按钮
```

#### 步骤 4：填写信息

**必填字段：**
- 工具名称 *（如：ChatGPT）
- 官方网站 *（如：https://chat.openai.com）
- 分类 *（选择对应分类）
- 价格类型 *（免费/付费等）
- 简短描述 *（50 字以内）

**选填字段：**
- 详细介绍（100-200 字）
- Logo URL
- 截图 URLs
- 标签（多个用逗号分隔）

#### 步骤 5：保存
```
点击 "保存" 按钮
选择 "立即发布" 或 "存为草稿"
```

## 📊 方式 3：批量导入

### 适用场景
- ✅ 一次性添加大量工具
- ✅ 从其他平台迁移
- ✅ 初始化数据

### 操作步骤

#### 准备 CSV 文件

**CSV 格式：**
```csv
name,url,category,priceType,shortDesc,description,tags
ChatGPT,https://chat.openai.com,AI 工具，freemium,AI 对话助手，ChatGPT 是...,AI;对话;生产力
Claude,https://claude.ai,AI 工具，freemium,AI 助手，Claude 是...,AI;对话;写作
```

**字段说明：**
- `name` - 工具名称（必填）
- `url` - 官网链接（必填）
- `category` - 分类（必填）
- `priceType` - 价格类型（必填）
- `shortDesc` - 简短描述（必填）
- `description` - 详细介绍（选填）
- `tags` - 标签，分号分隔（选填）

#### 导入步骤

**步骤 1：准备文件**
```
创建 CSV 文件
按格式填写工具信息
保存为 UTF-8 编码
```

**步骤 2：后台导入**
```
1. 登录后台
2. 进入 "工具管理"
3. 点击 "批量导入"
4. 上传 CSV 文件
5. 预览数据
6. 确认导入
```
