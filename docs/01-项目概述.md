## 🚀 技术选型

### 前端
| 技术 | 选型 | 理由 |
|------|------|------|
| 框架 | Next.js 14 | SSR/SEO 友好、React 生态 |
| UI 库 | Tailwind CSS + shadcn/ui | 快速开发、美观 |
| 状态管理 | Zustand | 轻量、易用 |
| 表单 | React Hook Form | 性能好、易用 |

### 后端
| 技术 | 选型 | 理由 |
|------|------|------|
| BaaS | Supabase | 开源 Firebase 替代、PostgreSQL |
| 认证 | Supabase Auth | 内置、支持多种登录方式 |
| 存储 | Supabase Storage | 图片/文件存储 |
| 实时 | Supabase Realtime | 实时数据同步 |

### 第三方服务
| 服务 | 用途 | 方案 |
|------|------|------|
| 搜索 | 全文搜索 | MeiliSearch / Algolia |
| 统计 | 访问分析 | Google Analytics + 百度统计 |
| 邮件 | 邮件发送 | SendGrid / 阿里云邮件推送 |
| 部署 | 网站托管 | Vercel（前端）+ Supabase（后端） |
| CDN | 静态资源 | Vercel Edge Network |
