## 💡 设计原则

1. **简洁即美** - 少即是多，避免过度设计
2. **一致性** - 统一的视觉语言贯穿全站
3. **层次感** - 通过阴影、大小、颜色建立层次
4. **动效克制** - 精致而非炫技，服务于功能
5. **无障碍** - 焦点环、对比度、键盘导航
6. **性能优先** - CSS 动画优于 JS，GPU 加速
