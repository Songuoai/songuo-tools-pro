#### Row Level Security (RLS)
```sql
-- tools 表
enable row level security on tools;

-- 所有人可读已发布的工具
create policy "Public tools are viewable by everyone"
  on tools for select
  using (status = 'published');

-- 仅管理员可修改
create policy "Admins can update tools"
  on tools for update
  using (auth.jwt()->>'role' = 'admin');

-- favorites 表
create policy "Users can view own favorites"
  on favorites for select
  using (auth.uid() = user_id);

create policy "Users can insert own favorites"
  on favorites for insert
  with check (auth.uid() = user_id);
```

### 搜索方案

#### 阶段一：前端搜索（Fuse.js）
```javascript
// 工具数据导出为 JSON
// public/tools-index.json

import Fuse from 'fuse.js'

const fuse = new Fuse(tools, {
  keys: ['name', 'description', 'tags'],
  threshold: 0.3,
  includeScore: true
})

const results = fuse.search('AI 音乐')
```

**优点：**
- 零成本
- 响应快
- 实现简单

**缺点：**
- 数据量大时性能下降
- 无拼音搜索
- 无错别字纠正

---

#### 阶段二：MeiliSearch
```javascript
// 自建 MeiliSearch 服务
import { MeiliSearch } from 'meilisearch'

const client = new MeiliSearch({
  host: process.env.MEILISEARCH_HOST,
  apiKey: process.env.MEILISEARCH_API_KEY
})

const index = client.index('tools')

// 搜索
const results = await index.search('AI 音乐', {
  filter: 'status = published',
  sort: ['click_count:desc']
})
```

**配置：**
```yaml
# docker-compose.yml
version: '3'
services:
  meilisearch:
    image: getmeili/meilisearch:latest
    ports:
      - "7700:7700"
    environment:
      - MEILI_MASTER_KEY=your_master_key
    volumes:
      - ./meili_data:/meili_data
```

**优点：**
- 搜索性能优秀
- 支持拼音搜索
- 支持错别字纠正
- 支持同义词

**成本：**
- 自建：服务器费用（约 ¥50/月）
- Cloud：MeiliSearch Cloud $20/月

### 认证安全
- Supabase Auth 内置安全
- JWT Token 验证
- RLS 行级权限控制
- CORS 配置

### 数据安全
- 输入验证（Zod）
- SQL 注入防护（参数化查询）
- XSS 防护（React 自动转义）
- CSRF 防护（Next.js 内置）

### 接口安全
- API Rate Limiting
- 敏感操作需要管理员权限
- Webhook 签名验证
