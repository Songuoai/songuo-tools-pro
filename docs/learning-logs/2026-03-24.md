## 💡 学习收获

### 1. Tailwind CSS 真好用！
```
传统 CSS：写样式要切换文件
Tailwind：直接在 HTML 中写类名，效率超高

示例：
<div className="bg-white rounded-xl shadow-md p-6 hover:shadow-xl transition">
快速实现：白色背景 + 圆角 + 阴影 + 内边距 + hover 效果
```

### 2. 组件化开发
```
首页 = 导航栏 + Hero + 分类 + 推荐 + 页脚
每个部分独立开发，最后组装
```

### 3. 响应式设计
```
手机：grid-cols-1（1 列）
平板：md:grid-cols-2（2 列）
电脑：lg:grid-cols-4（4 列）
```

## 🐛 遇到的问题

### 问题 1：渐变背景语法
**现象：** 不知道如何写渐变
**解决：** 查 Tailwind 文档，使用 bg-gradient-to-b 和 from-xxx to-xxx
**收获：** 学会了 Tailwind 渐变语法

### 问题 2：hover 动画
**现象：** 卡片 hover 时想要上浮效果
**解决：** 使用 hover:-translate-y-1 + transition
**收获：** 学会了 Tailwind 动画类
