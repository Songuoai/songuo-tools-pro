## 🐛 遇到的问题

### 问题 1：动态路由
**现象：** 不知道如何创建动态详情页
**解决：** 学习 Next.js 动态路由，使用 [slug] 文件夹
**收获：** 掌握了 Next.js 动态路由系统

### 问题 2：选项卡切换
**现象：** 需要实现介绍/评价/类似的切换
**解决：** 使用 useState 管理 activeTab 状态，条件渲染内容
**收获：** 学会了 React 状态管理和条件渲染

### 问题 3：模拟数据
**现象：** 后端还没开发，需要展示数据
**解决：** 创建 mock 数据对象，模拟真实数据结构
**收获：** 学会了前后端分离开发模式
