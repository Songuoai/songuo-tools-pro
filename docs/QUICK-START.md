## 🛠️ 第三步：添加工具（重点）

### 进入工具管理
```
点击左侧菜单：工具管理
```

### 添加工具
```
点击右上角：添加工具 按钮
```

### 填写信息

**必填（5 项）：**
1. 工具名称 * 
   - 例：ChatGPT
   
2. 官方网站 *
   - 例：https://chat.openai.com
   
3. 分类 *
   - 选择：AI 工具
   
4. 价格类型 *
   - 选择：🎁 免费额度
   
5. 简短描述 *
   - 例：OpenAI 开发的 AI 对话助手

**选填（3 项）：**
6. 详细介绍
   - 100-200 字完整介绍
   
7. Logo URL
   - Logo 图片链接
   
8. 标签
   - 例：AI, 对话，生产力

### 保存发布
```
发布状态：选择 "发布"
点击：保存工具
```

## 📝 常用操作

### 编辑工具
```
1. 工具管理
2. 找到要编辑的工具
3. 点击"编辑"按钮
4. 修改信息
5. 保存
```

### 删除工具
```
1. 工具管理
2. 找到要删除的工具
3. 点击"删除"按钮
4. 确认删除
```

### 审核投稿
```
1. 投稿审核
2. 查看投稿列表
3. 点击"通过"或"拒绝"
```
