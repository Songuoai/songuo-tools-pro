# 松果工具箱 Pro - 功能规划

## 📱 前台用户端功能

### 1. 首页 (index.html)

**核心模块：**

| 模块 | 功能描述 | 优先级 |
|------|---------|-------|
| 顶部导航 | Logo、分类菜单、搜索框、用户登录 | P0 |
| Hero 区域 | 网站 Slogan、搜索框、热门分类快捷入口 | P0 |
| 分类统计 | 6 大分类卡片（工具数、图标） | P0 |
| 热门推荐 | 精选工具卡片（6-12 个） | P0 |
| 分类展示 | 各分类工具列表（带标签筛选） | P0 |
| 专题推荐 | 运营策划专题（3-5 个） | P1 |
| 工具提交入口 | "提交工具"按钮 | P1 |
| 页脚 | 版权信息、友情链接、备案号 | P0 |

**交互细节：**
- 搜索框支持实时建议
- 工具卡片 hover 效果

- 移动端汉堡菜单

### 7. 工具提交页 (/submit)

**提交流程：**

```
1. 填写基本信息
   - 工具名称 *
   - 官方链接 *
   - 工具简介 * (200 字内)

2. 选择分类
   - 主分类 *
   - 子分类
   - 标签 (3-5 个) *

3. 补充信息
   - 功能特点
   - 使用场景
   - 截图上传（可选）

4. 提交者信息
   - 姓名/昵称
   - 邮箱 *
   - 备注（可选）

5. 提交审核
   - 同意协议
   - 提交 → 等待审核 → 通知结果
```

## 🔧 核心功能模块

### 1. 搜索系统

**技术方案：**
```
方案 A（初期）：前端搜索
- 工具数据导出为 JSON
- 使用 Fuse.js 进行前端模糊搜索
- 优点：简单、零成本
- 缺点：数据量大时性能下降

方案 B（后期）：MeiliSearch
- 自建 MeiliSearch 服务
- 实时索引工具数据
- 支持拼音搜索、错别字纠正
- 优点：性能好、功能强
- 成本：服务器费用
```

**搜索功能：**
- 关键词搜索
- 分类筛选
- 标签筛选
- 排序（相关度、热门、最新）
- 搜索建议
- 热门搜索
